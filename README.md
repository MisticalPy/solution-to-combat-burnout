# Проект: Борьба с выгоранием сотрудников через Telegram-бота

Данный проект предназначен для мониторинга состояния сотрудников и предотвращения их выгорания с помощью интегрированного Telegram-бота. Бот осуществляет анализ на основе данных, хранящихся в базе данных, а также информации, полученной от пользователей.

## Как запустить проект

Следуйте инструкциям ниже для запуска проекта на вашей машине.

### 1. Запуск Docker-контейнеров

Перейдите в директорию, содержащую файл `docker-compose.yml`, и выполните команду:
```bash
docker compose up
```
Подождите, пока не завершится установка всех необходимых пакетов и контейнеров.

### 2. Создание виртуального окружения и его активация

**На Windows** выполните следующую команду для создания виртуального окружения:
```cmd
python -m venv venv
```

А затем активируйте его командой:
```cmd
venv\Scripts\activate
```

### 3. Установка зависимостей

После активации виртуального окружения установите все необходимые библиотеки командой:
```bash
pip install -r requirements.txt
```

### 4. Запуск основного файла

Перейдите в директорию с файлом `main.py` и запустите его:
```bash
python main.py
```

## Использование бота

После запуска, откройте Telegram и перейдите по следующей ссылке, чтобы начать взаимодействие с ботом:  
[@testing_order_avito_bot](https://t.me/testing_order_avito_bot)
